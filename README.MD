---

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+)
- [MongoDB](https://www.mongodb.com/) instance

### Installation & Local Setup

1. **Clone the Repository**
   ```bash
   git clone https://github.com/your-username/flyrank-ai.git
   cd capstone
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Environment Setup**
   Create a `.env` file in the root directory:
   ```env
   PORT=5000
   MONGO_URI=mongodb://localhost:27017/flyrank
   ```

4. **Run Development Server**
   ```bash
   # Start Frontend
   npm run dev
   ```
